- Debounce Search
- Sorting
- Filtering
- Caching

## Installation

- Clone the repository

```bash
 git clone https://github.com/SikandarJODD/youva
```

- Install the dependencies

```bash
 pnpm install
```

#### [Checkout Usage & Learning](/usage.md)

- Run the project

```bash
 pnpm dev
```
